# Pakistan Single Window Trade Operations Reference

## Source

- Pakistan Single Window: `https://www.psw.gov.pk`
- Pakistan Customs and FBR cross-border trade processing references where linked through PSW

## Common Workflow Surfaces

- import declarations
- export declarations
- supporting document submission
- regulatory agency approvals
- customs-linked processing

## Key Distinctions

- PSW process layer vs customs tariff schedule
- document submission vs substantive duty/exemption determination
- platform workflow vs port cargo operations

## Extraction Notes

- Preserve whether the issue is process, document, or agency-coordination related.
- If the task becomes HS-code or duty-rate specific, hand off to `fbr-customs-tariff`.
