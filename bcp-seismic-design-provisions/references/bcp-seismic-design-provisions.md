## Document History

- Trigger event: the October 2005 Kashmir earthquake exposed the absence of a dedicated Pakistani seismic design code.
- Development: Pakistan Engineering Council convened a task force with academia, industry, Ministry of Housing & Works, Ministry of Science and Technology, and NESPAK, with international technical collaboration from the International Code Council (ICC) and American Concrete Institute (ACI).
- Approval: Cabinet approval on 9 August 2007.
- Legal notification: Government of Pakistan S.R.O. 971(I)/2008, dated 8 September 2008.
- Successor/expansion: the 2021 Building Code of Pakistan, published in ICC's Digital Codes platform (document ID PKBC2021P1), is a more recent and broader code compilation that carries forward seismic-resilience requirements alongside fire/life-safety and general building provisions, including an appendix specifically on seismic evaluation and retrofit of existing buildings.

## Key Concepts (Conceptual Framework Only — Verify Current Numeric Values Against Official Text)

- **Seismic zoning for design**: the code divides the country into design seismic zones based on expected ground-motion intensity, distinct from PMD's real-time hazard/event monitoring products. Zone boundaries and factors must be read from the current official code/map, not assumed from older secondary sources.
- **Design response spectrum methodology**: conceptually combines a seismic zone factor, a site (soil) classification, an importance factor tied to occupancy/criticality of the building, and a response modification (ductility-related) factor tied to the chosen structural system, to derive a design base shear or spectral acceleration.
- **Analysis procedures**: equivalent static lateral force procedure for regular, lower-rise structures; dynamic analysis (response spectrum or time-history) required for taller, irregular, or higher-importance structures per code-specified applicability thresholds.
- **Ductile detailing**: seismic design categories drive minimum ductile-detailing requirements for reinforced concrete and steel members and connections, distinct from ordinary (non-seismic) detailing.
- **UBC-97 lineage**: BCP-SP 2007 was developed drawing on the 1997 Uniform Building Code seismic methodology, which Pakistani practice referenced prior to 2007; comparative literature (e.g., the Springer article above) discusses differences in outcomes between BCP-2007 and UBC-97 application.

## Scope Exclusions (BCP-SP 2007)

Explicitly excluded from BCP-SP 2007 coverage: bridges, dams, tunnels, pipelines, power plants, defense installations, and underground structures. These require other design standards/authorities (e.g., highway bridges under NHA standards, dams under WAPDA/dam-safety engineering practice).

## Extraction Notes

- Always state which code edition (BCP-SP 2007 vs. 2021 Building Code of Pakistan) a cited provision belongs to.
- Do not restate specific seismic zone factors, response spectrum ordinates, importance factors, or response modification coefficients as current fact without verifying against the live code text (PEC or ICC Digital Codes); these values are the kind of detail that gets revised between editions.
- When secondary/academic sources (e.g., comparative BCP-2007-vs-UBC-97 papers) are used for background, label them clearly as secondary commentary, not the primary code text.
- This code is a life-safety design standard; outputs from this skill support research and drafting only and are not a substitute for a licensed structural engineer's design and sign-off.
