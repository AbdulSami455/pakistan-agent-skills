# IRSA Reservoir Data Reference

## Source

- Indus River System Authority (IRSA) Daily Data page: `http://pakirsa.gov.pk`
- Note: `irsa.gov.pk` is not the authority's correct operating domain for this data; always use `pakirsa.gov.pk`. The site does not serve valid HTTPS — use the `http://` scheme.

## Reservoirs and Barrages Covered

- Tarbela (on the Indus River)
- Mangla (on the Jhelum River)
- Chashma (on the Indus River, a barrage with limited storage)
- Other key barrages may appear in flow tables (e.g., Taunsa, Guddu, Sukkur, Kotri) depending on the bulletin format.

## Typical Daily Bulletin Fields

- Date
- Reservoir level (feet)
- Live storage (acre-feet)
- Inflow (cusecs)
- Outflow/discharge (cusecs)
- Same-date-last-year comparison figures
- Minimum operating level / dead storage level reference

## Unit Definitions

- Level: elevation in feet above a reference datum.
- Storage: volume in acre-feet (1 MAF = 1,000,000 acre-feet).
- Flow (inflow/outflow/discharge/withdrawal): rate in cusecs (cubic feet per second).

## Extraction Notes

- Always anchor any figure to its exact date; daily data changes day to day and is not representative of monthly or seasonal averages without aggregation.
- IRSA's daily data is operational and real-time; it does not include policy narrative, investment planning, or annual water-sector commentary.
