# Pakistan Earthquake and Seismic Hazard Reference

## Source

- Pakistan Meteorological Department seismology and earthquake bulletins: `https://www.pmd.gov.pk`
- Geological Survey of Pakistan geoscience references: `https://www.gsp.gov.pk`
- NDMA hazard and preparedness material where seismic zoning is referenced: `https://ndma.gov.pk`

## Typical Event Fields

- date and origin time
- latitude and longitude
- magnitude
- depth
- epicenter / nearest named area
- felt-region note where provided

## Common Context Layers

- Hindukush deep-focus seismicity
- Chaman fault system
- Kashmir and northern mountain belt seismic exposure
- Balochistan seismicity

## Extraction Notes

- Preserve whether the bulletin is an event notice, a daily catalog, or a hazard-context document.
- If the source names a magnitude scale, keep it attached to the value.
- Do not infer damage from a magnitude value alone.
