## Accreditation Status Types

- **Accredited / Fully Accredited** — program meets all council standards; valid for the stated period.
- **Provisional Accreditation** — program is new or under review; granted for a limited time with conditions.
- **Accredited with Conditions** — program meets most standards but must address specific deficiencies.
- **Not Listed / Denied** — program is not accredited by the council; graduates may not be eligible for professional registration.
- **Expired** — accreditation was granted but the validity period has lapsed without renewal.

## Extraction Notes

- Always check the council's own website directly — HEC's QAA page links to councils but may not have real-time status updates.
- PMC replaced PMDC in October 2020 — records before that date reference PMDC; current queries should use PMC.
- Accreditation is program- and campus-specific: a university's main campus may be accredited while its sub-campus is not.
- Some programs may be "registered" but not "accredited" — registration is a lower bar; only accredited programs qualify for professional engineering/medical registration.
- NBEAC uses a tiered system (W, X, Y, Z categories) — preserve the exact category letter when citing NBEAC accreditation.
- NCEAC uses a tiered system (NCEAC-1, NCEAC-2, etc.) — preserve the tier exactly.
