# Provincial Social Security Employer Registration Reference

## Source

- Provincial Employees Social Security Institutions / labour welfare departments
- Province-specific social security acts, rules, and contribution guidance

## Common Institutions

- Punjab Employees Social Security Institution (PESSI)
- Sindh Employees Social Security Institution (SESSI)
- equivalent provincial labour/social security bodies in other provinces where applicable

## Common Compliance Surfaces

- employer registration
- employee enrollment
- contribution payment
- coverage records and benefit-linked documentation

## Extraction Notes

- Always attach guidance to the relevant province because labour welfare administration is not nationally uniform.
- Distinguish provincial social security from EOBI and from workers welfare fund obligations.
