# IPO Pakistan Trademarks Reference

## Source

- Intellectual Property Organization of Pakistan: `https://ipo.gov.pk`
- Trademarks Registry materials, forms, procedural notes, and public notices where available

## Common Workflow Stages

- pre-filing search
- filing
- examination / objection
- publication
- opposition
- registration
- renewal

## Key Distinctions

- SECP company name vs trademark right
- goods classes vs services classes
- word mark vs logo/device mark where the source distinguishes them

## Extraction Notes

- Preserve whether the task is about clearance, process stage, or rights maintenance.
- Trademark process stages should be kept separate because each has different office actions and timelines.
