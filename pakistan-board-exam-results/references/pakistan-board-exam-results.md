# Pakistan Board Exam Results Reference

## Source

- Federal and provincial Boards of Intermediate and Secondary Education official result portals and notifications
- Board-issued gazettes, press releases, and annual result announcements
- Inter Board Committee of Chairmen context where cross-board coordination material is relevant

## Common Result Dimensions

- board name
- SSC vs HSSC
- Part-I vs Part-II
- annual vs supplementary / second annual
- appeared, passed, and pass percentage
- top positions / merit positions

## Common Document Types

- result gazette
- press release
- notification
- revised or corrected result notice

## Extraction Notes

- Keep candidate-level gazette data distinct from board-level summary notifications.
- Preserve the board's exact exam-cycle terminology.
- If a board renames supplementary exams as second annual exams, quote the board's wording rather than normalizing silently.
